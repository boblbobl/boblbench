boblbench
Workbench-inspired blog terminal

boblbench:Workbench

This is the start of a browser-native desktop environment built as a public-facing blog.

The visual language now leans toward Amiga Workbench 1.0: muted teal backdrop, blue title bars, orange accents, thick bevels, and icon-driven navigation.

The plan is simple: make it feel like booting into a strange little personal machine full of notes, projects, experiments, and digital artifacts.
