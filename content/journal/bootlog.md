# Bootlog

A first pass at a **Workbench-inspired** web desktop.

## Milestones

- Static prototype established
- Visual language shifted toward Amiga Workbench 1.0
- Drawers and files now define the site structure
- Markdown viewer added as the first real application

## Why markdown?

Because it gives the site a nice split between:

1. old-school window chrome
2. modern readable writing

> The desktop is the shell. Markdown is the content.
